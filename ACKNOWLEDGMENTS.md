# Acknowledgments

Brain Researcher builds on open scientific data standards, neuroimaging tools,
knowledge resources, and infrastructure projects.

## Data, Standards, And Ontologies

- [OpenNeuro](https://openneuro.org/)
- [BIDS](https://bids.neuroimaging.io/)
- [Cognitive Atlas](https://www.cognitiveatlas.org/)
- [NeuroSynth](https://neurosynth.org/)
- [NeuroBagel](https://neurobagel.org/)
- [Allen Brain Atlas](https://portal.brain-map.org/)

Dataset-specific licenses and access restrictions remain with the upstream data
providers. Brain Researcher demo bundles are curated report artifacts unless a
release gate explicitly states that the underlying workflow was freshly executed
in hosted production.
