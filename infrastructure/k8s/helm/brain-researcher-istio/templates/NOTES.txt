## Troubleshooting

### Common Commands
```bash
# Check sidecar injection
istioctl analyze

# View service mesh configuration
istioctl proxy-config cluster <pod-name> -n {{ include "brain-researcher-istio.namespace" . }}

# Debug traffic routing
istioctl proxy-config routes <pod-name> -n {{ include "brain-researcher-istio.namespace" . }}

# Check security policies
istioctl authn tls-check <service-name>.<namespace>.svc.cluster.local

# View telemetry configuration
kubectl get telemetry -n {{ include "brain-researcher-istio.namespace" . }} -o yaml
```

### Validation
Run the following to validate your setup:
```bash
# Test internal service communication
kubectl exec -it <pod-name> -n {{ include "brain-researcher-istio.namespace" . }} -- curl -v http://<service-name>:8080/health

# Check mTLS status
istioctl authn tls-check <service-name>.<namespace>.svc.cluster.local

# Verify gateway configuration
istioctl analyze --namespace {{ include "brain-researcher-istio.namespace" . }}
```
