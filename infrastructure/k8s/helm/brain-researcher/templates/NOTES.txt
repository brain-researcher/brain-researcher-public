🔧 Troubleshooting
================================================================================

If you encounter issues:

1. 🔍 Check pod logs:
   $ kubectl logs -n {{ include "brain-researcher.coreNamespace" . }} -l app.kubernetes.io/name={{ include "brain-researcher.name" . }}

2. 📋 Describe failing pods:
   $ kubectl describe pods -n {{ include "brain-researcher.coreNamespace" . }}

3. 🔍 Check for ImagePullBackOff:
   $ kubectl get events -n {{ include "brain-researcher.coreNamespace" . }} --sort-by='.lastTimestamp'

4. 🔐 Verify secrets are mounted correctly:
   $ kubectl get secrets,configmaps -n {{ include "brain-researcher.coreNamespace" . }}

5. 🌐 Test network connectivity between services:
   $ kubectl run -it --rm debug --image=busybox --restart=Never -n {{ include "brain-researcher.coreNamespace" . }} -- sh
   # Then inside the pod:
   # wget -O- http://{{ include "brain-researcher.fullname" . }}-br-kg:5000/health

Common issues and solutions:
- ImagePullBackOff: Verify image exists in registry ({{ .Values.global.imageRegistry }})
- CrashLoopBackOff: Check logs for missing dependencies or configuration errors
- Pending pods: Check storage class availability and node resources
- StatefulSet not updating: Delete pod manually to force recreation with new image

Force pod recreation for StatefulSets:
$ kubectl delete pod {{ include "brain-researcher.fullname" . }}-br-kg-0 -n {{ include "brain-researcher.coreNamespace" . }}
$ kubectl delete pod {{ include "brain-researcher.fullname" . }}-agent-0 -n {{ include "brain-researcher.coreNamespace" . }}

  --set global.imageTag=$TAG \
  -n {{ include "brain-researcher.coreNamespace" . }}

🎯 Architecture Notes:
- Unified Gateway: Agent service handles both HTTP (/api) and WebSocket (/ws) on port 8000
- Metrics: Exposed on main service ports (/metrics), no separate metrics ports
- Sessions: Stored in Redis, with persistent StatefulSet storage for agent
- WebSocket: Real-time updates available at /ws/dashboard, /ws/jobs/{id}, etc.

🎯 Pro Tips:
- Use `kubectl top pods -n {{ include "brain-researcher.coreNamespace" . }}` to monitor resource usage
- StatefulSets require manual pod deletion to pick up new images
- Check Artifact Registry for available image tags before deployment
- Monitor SSL certificate provisioning in GCP Console (can take 10-15 minutes)
- Review and update security policies regularly
