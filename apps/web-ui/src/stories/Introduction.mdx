# Brain Researcher UI Component Library

Welcome to the Brain Researcher UI component library documentation. This Storybook contains all the components, design tokens, and guidelines for building consistent user interfaces in the Brain Researcher platform.

## What is Brain Researcher?

Brain Researcher is a neuroimaging analysis platform that combines LLM-powered natural language understanding with fMRI analysis tools, knowledge graphs, and cognitive modeling. Our UI components are designed to handle complex scientific data visualization and analysis workflows.

## Key Features

### 🧠 Neuroimaging Focused
Components designed specifically for brain imaging data visualization and analysis workflows.

### ♿ Accessibility First
All components follow WCAG 2.1 AA guidelines with keyboard navigation, screen reader support, and high contrast modes.

### 📱 Responsive Design
Mobile-first approach ensuring usability across devices from phones to large desktop displays.

### 🎨 Theme Support
Full dark/light mode support with customizable design tokens for different research contexts.

### 🔧 Developer Experience
TypeScript support, comprehensive documentation, and interactive examples for rapid development.

## Component Categories

- **UI Primitives**: Basic building blocks (Button, Input, Card, etc.)
- **Data Visualization**: Charts, graphs, and brain visualization components
- **Scientific Tools**: Domain-specific components for neuroimaging analysis
- **Navigation**: Headers, menus, and routing components
- **Layout**: Grid systems, containers, and responsive layouts
- **Feedback**: Loading states, error handling, and user notifications
